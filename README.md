# Dashboard de Escala Operacional

Projeto desenvolvido para automatizar a identificação de coberturas operacionais (REDE) em escalas de motoristas.

## Objetivo

Eliminar a necessidade de consulta manual da escala e identificar rapidamente quais turnos necessitam cobertura em uma determinada data.

## Tecnologias

- Python
- Pandas
- Google Colab

## Funcionalidades

- Leitura automática de planilha Excel
- Mapeamento de datas da escala
- Consulta de coberturas por data
- Filtragem por operação
- Retorno estruturado em DataFrame

## Exemplo

Data: 2026-06-15

Santa Adélia | 1ºT | Motorista ficticio
Araraquara | 1ºT | Vaga aberta

Total de redes: 2
